### Performance
- Lazy load all images with `loading="lazy"`
- Font display: `font-display: swap` in all `@font-face` or Google Fonts URLs
- Use `will-change: transform` only on actively animating elements, remove after animation completes

### Accessibility
- All interactive elements have visible focus states (`outline: 2px solid [accent-color]`)
- Images have meaningful `alt` text
- Form inputs have associated `<label>` elements
- Navbar has `role="navigation"` and `aria-label`
- Mobile hamburger button has `aria-expanded` and `aria-label`

### Scroll Animations (Both Projects)
- Use Framer Motion `whileInView` with `viewport={{ once: true, margin: "-80px" }}`
- Never replay on scroll-up — `once: true` on all entrance animations
- Default entrance: `y: 30 → 0, opacity: 0 → 1`
- A&M Studios duration: `0.7s–0.9s` (slower, dreamier)
- Ember & Oak duration: `0.2s–0.3s` (faster, snappier)

### Mobile Breakpoints
```
480px  — small phones
768px  — tablet / hamburger menu trigger
968px  — mid-size layout adjustments
1200px — max content width clamp
```

### Smooth Scroll
- `html { scroll-behavior: smooth; }`
- Navbar links use `href="#section-id"` anchors
